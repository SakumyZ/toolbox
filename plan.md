## 新增功能规划

### 一、系统工具类

#### 1. Windows 服务管理器 ⭐ 优先级：高

> 用户明确需求

**功能：**

- 列出所有 Windows 服务（名称、状态、启动类型、描述）
- 启动 / 停止 / 重启服务
- 修改启动类型（自动 / 手动 / 禁用）
- 搜索 + 筛选（按状态、按名称关键字）
- 常用服务置顶/收藏（如 MySQL、Redis、IIS、Docker Desktop Service）
- 一键批量启停（场景化：如"启动开发环境"一键启动 MySQL + Redis + Nginx）

**技术要点：**

- `System.ServiceProcess.ServiceController` API
- 需要管理员权限（启停操作时提权）
- 实时状态轮询或 WMI 事件监听

---

#### 2. 端口占用查看器 ⭐ 优先级：高

> 开发常见痛点：端口被占用不知道是哪个进程

**功能：**

- 列出所有监听中的 TCP/UDP 端口
- 显示占用进程（PID、进程名、路径）
- 搜索特定端口号
- 一键结束占用进程
- 常用端口高亮标记（80, 443, 3000, 3306, 5432, 6379, 8080...）

**技术要点：**

- `netstat` / `GetExtendedTcpTable` Win32 API
- `Process.GetProcessById` 获取进程信息

---

#### 3. Hosts 文件编辑器 ⭐ 优先级：中

> 开发经常需要切换 hosts 配置

**功能：**

- 可视化编辑 `C:\Windows\System32\drivers\etc\hosts`
- 单行启用/禁用（注释切换）
- 分组管理（如：开发环境、测试环境、生产环境）
- 方案切换（一键切换整组 hosts 配置）
- 语法校验 + 重复检测

**技术要点：**

- 文件读写需要管理员权限
- 备份/恢复机制

---

#### 4. 环境变量管理器 ⭐ 优先级：中

> 比系统自带的环境变量编辑器好用 100 倍

**功能：**

- 分区显示：用户变量 / 系统变量
- PATH 变量特殊处理：拆分为列表，支持拖拽排序
- 搜索变量名/值
- 新增 / 编辑 / 删除变量
- 导出/导入配置（方便迁移电脑）
- 变量值路径有效性检测（标红失效路径）

**技术要点：**

- `Environment.GetEnvironmentVariable` + 注册表读写
- 修改系统变量需要管理员权限
- 广播 `WM_SETTINGCHANGE` 使修改立即生效

---

### 二、开发工具类

#### 5. JSON / 数据格式化工具 ⭐ 优先级：高

> 每天都在用的工具，不用再开网页了

**功能：**

- JSON 格式化 / 压缩 / 校验
- JSON ↔ C# Class 互转
- JSON Path 查询
- Base64 编码/解码
- URL 编码/解码
- 时间戳 ↔ 日期时间 转换
- UUID/GUID 生成
- Hash 计算（MD5、SHA1、SHA256）

**技术要点：**

- `System.Text.Json` 处理 JSON
- 纯本地计算，无需网络

---

#### 6. 正则表达式测试器 ⭐ 优先级：中

**功能：**

- 输入正则 + 测试文本，实时高亮匹配结果
- 显示捕获组
- 常用正则模板库（邮箱、手机号、IP、URL 等）
- 正则语法解释

**技术要点：**

- `System.Text.RegularExpressions.Regex`
- 实时匹配 + 超时保护（防止 ReDoS）

---

#### 7. 颜色拾取器 ⭐ 优先级：低

**功能：**

- 屏幕取色（全局快捷键触发）
- 颜色格式转换（HEX ↔ RGB ↔ HSL ↔ CMYK）
- 调色板
- 常用颜色收藏
- 历史取色记录

**技术要点：**

- `GetPixel` Win32 API 或全局鼠标 Hook
- 截图放大镜效果

---

### 三、效率工具类

#### 8. 剪贴板历史管理 ⭐ 优先级：高

> Windows 自带的 Win+V 功能太弱

**功能：**

- 自动记录剪贴板历史（文本、图片、文件路径）
- 搜索历史内容
- 置顶/收藏常用条目
- 全局快捷键呼出（如 Ctrl+Alt+V）
- 分类过滤（文本/代码/链接/图片）
- 支持设置保留条数上限

**技术要点：**

- `Clipboard` 监听（`AddClipboardFormatListener` Win32 API）
- SQLite 持久化
- 全局热键复用 `HotkeyService`

---

#### 9. 快捷启动器 / 命令面板 ⭐ 优先级：中

> 类似 Alfred / Raycast 的 Windows 本地版

**功能：**

- 全局快捷键呼出（Ctrl+Space 或自定义）
- 搜索并启动应用程序
- 搜索文件/文件夹
- 自定义快捷命令（如：打开 VS Code 到指定项目）
- 内置计算器（输入表达式直接算）
- 搜索 ToolBox 内所有工具

**技术要点：**

- 遍历 Start Menu 快捷方式 + PATH 下可执行文件
- `Process.Start` 启动程序

---

#### 10. 文件批量重命名 ⭐ 优先级：低

**功能：**

- 拖拽文件/文件夹进来
- 规则重命名：前缀/后缀、查找替换、序号、日期
- 实时预览重命名结果
- 撤销（记住原名）

---

### 四、系统信息类

#### 11. 系统信息面板（增强 Dashboard）⭐ 优先级：中

> 把 Dashboard 做成真正有用的总览页

**功能：**

- CPU / 内存 / 磁盘 实时使用率（图表）
- 网络速度监控
- 系统启动时间
- .NET / Node.js / Python / Java 版本检测
- 已安装开发工具版本一览
- 快捷入口：打开各工具页

**技术要点：**

- `PerformanceCounter` 或 WMI 查询
- `DispatcherTimer` 定时刷新

## 技术注意事项

1. **管理员权限**：服务管理、Hosts 编辑、环境变量修改需要提权。建议在 `app.manifest` 中按需请求，或运行时动态提权。
2. **全局热键**：已有 `HotkeyService` 基础设施，新增工具（剪贴板、启动器）复用即可。
3. **SQLite**：已有 `DatabaseService`，剪贴板历史等新功能共用数据库。
4. **性能**：系统信息面板的实时刷新注意控制频率，避免 CPU 开销过大。
5. **一致性**：所有新页面遵循现有的三栏/卡片式布局风格，保持 UI 一致性。
